# Learnist

A learnist is an all in one open source platform to manage your studies, homeworks, etc.

## Features

- 🚀 Open Source
- 🌐 Easy to use
- 🕶️ Dark mode
- 💻 Cross platform

> Made as part of the Hashnode x PlanetScale Hackathon
